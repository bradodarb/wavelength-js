Micro-framework for FAAS applications.

## Install

```shell
yarn add wavelength-js
```


## Import package into your application

### TypeScript

```typescript
import {Runtime} from "wavelength-js";
```

### JavaScript

```javascript
const {Runtime} = require("wavelength-js");
```


## commands

`yarn run build`

`yarn run watch`

`yarn run test`
